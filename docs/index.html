PNG Concat

A streaming PNG concatenation library for Node.js and web browsers

Live Examples & Documentation

Try out the library's features right in your browser! Each example shows the code and lets you run it to see the results.

Features: No Canvas Required • Streaming Output • Zero Dependencies • Type Safe • Flexible Layouts

1. Horizontal Concatenation

Combine multiple images side by side in a single row.

const result = await concatPngs({ inputs: [png1, png2, png3], layout: { columns: 3 } // 3 images in a row });

Source Images:

2. Vertical Concatenation

Stack multiple images vertically in a single column.

const result = await concatPngs({ inputs: [png1, png2, png3], layout: { rows: 3 } // 3 images stacked vertically });

Source Images:

3. Grid Layout

Arrange images in a grid pattern with multiple rows and columns.

const result = await concatPngs({ inputs: [png1, png2, png3, png4, png5, png6], layout: { columns: 3 } // 2 rows × 3 columns grid });

Source Images:

4. Arbitrary Image Sizes

Mix images of different dimensions - smaller images are automatically padded with transparency.

const result = await concatPngs({ inputs: [small, medium, large], layout: { columns: 3 } // Automatically handles different sizes with transparent padding });

Source Images (different sizes):

5. Width Limit with Wrapping

Limit the output width - images automatically wrap to new rows when the limit is reached.

const result = await concatPngs({ inputs: [png1, png2, png3, png4], layout: { width: 100 } // Max 100px wide, wraps automatically });
